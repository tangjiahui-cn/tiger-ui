# tiger-ui
一个轻量化、简洁的React UI库。(开发中)

## 技术栈
```text
React18 + TypeScript + Webpack5 + Less
```

## 如何运行
``` shell
pnpm install

pnpm dev

pnpm build
```

## 已支持特性
```text
- 支持国际化
- 支持全局化配置
- 打包单独的组件（支持按需引入）
- 拥有完整的单元测试（3/n）
```
