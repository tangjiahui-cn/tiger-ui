# tiger-ui
一个轻量化、简洁的React UI库。(重构中)

## 技术栈
```text
React18 + TypeScript + Webpack5 + class-css
```
## 如何运行
``` shell
pnpm install

pnpm dev

pnpm build

// 运行文档
pnpm docs
```

## 已支持特性
```text
- 支持国际化
- 支持全局主题色配置
- 拥有完整的单元测试（3/n）
- 拥有完整的docs站点
- 无需引入css文件（使用css-in-js）
- 导出UMD、CJS、ESM包
```
