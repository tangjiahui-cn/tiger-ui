### 快速开始
使用 npm。
```shell
npm install tiger-ui
```
使用 yarn。
```shell
yarn add tiger-ui
```
使用 pnpm。
```shell
pnpm add tiger-ui
```
> 推荐使用 pnpm 进行包依赖管理，其可以避免幽灵依赖问题，以及更好的节省计算机存储空间。

### 运行代码
如果你使用过antd，那么你将很快上手。
```jsx | pure
import { Button } from 'tiger-ui';

export default () => {
  return (
    <Button>按钮</Button>
  )
}
```
